    How it works

    The front end runs on this static site; the CV and job description are sent to a small serverless function (a Cloudflare Worker) that holds the AI key securely and calls Google's Gemini model. Keeping the key server-side is the correct, safe pattern — it's never exposed in the browser.

    Stack: vanilla JavaScript front end · Cloudflare Worker backend · Google Gemini API · deployed free.

    Privacy & AI notice: This tool uses AI. Text you enter is sent to Google’s Gemini API to generate a response and is not stored by this site. Please don’t paste confidential or personal information. For demonstration purposes only — not career, legal or professional advice.
