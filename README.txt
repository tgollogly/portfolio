THOMAS GOLLOGLY — PORTFOLIO SITE
================================
Open index.html. Keep all files together in this folder.

  index.html            homepage + CV links + AI chatbot (💬 bottom-right)
  cv.html               your CV (printable to PDF)
  ats-matcher.html      AI ATS Resume Matcher tool
  heat-dome.html        live demo + case study
  thermal-compare.html  live demo + case study
  beneish.html          live demo + case study
  bundlebuilder.html    live demo + case study
  config.js             <- paste your AI backend URL here (see AI-SETUP.md)
  backend/worker.js     the serverless backend for the AI features
  AI-SETUP.md           step-by-step: switch on the chatbot + ATS tool (free)

DEPLOY THE SITE (free): create a GitHub repo named  yourusername.github.io
and upload everything EXCEPT the backend folder (that goes to Cloudflare).
Or drag the folder onto https://app.netlify.com/drop

OPTIONAL: add GITHUB_URL and LINKEDIN_URL in index.html (find "EDIT ZONE").
