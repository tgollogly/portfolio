Selected projects — all live, try them from my portfolio

AI ATS Resume Matcher — full-stack + AI
A full-stack AI tool: score a CV against any job description, see the missing keywords, then generate a tailored cover letter and an improved CV to download. Built on a secure serverless backend (Cloudflare Worker) that holds the API key and calls Google’s Gemini model — the browser never sees the key.
BOM Desk
Data administration console for manufacturing bills of materials. Holds material and labour lines, runs eight validation rules live, rejects invalid rows at import, exports CSV reports and logs every change. Built with invented demo data — not a deployment of any live system.
SQL Lab
A real SQLite engine compiled to WebAssembly and run client-side over a five-table relational schema. Schema browser, free-form query editor and eleven worked examples covering joins, aggregation, subqueries, CTEs and window functions, with CSV export of any result set.
Test Bench
A TypeScript validation library (UK postcodes, sort codes, IBAN mod-97 checksums, strict date parsing, integer-pence money handling, CSV escaping) with a 28-case unit test suite. Library and suite both compile and execute in the browser; a deliberate-bug button demonstrates the suite catching a regression and naming the failing case.
Heat Anomaly Detector
Live heat-risk dashboard that compares today against the 30-year climate average (WMO 1991–2020) to flag real heat anomalies: real-time weather, a 3D terrain map, live radar and an auto-refresh watch mode. All key-free and client-side.
Beneish M-Score Screener
Runs the full eight-factor forensic-accounting model in the browser from two years of company figures to flag earnings-manipulation risk — no data leaves the device.
Thermal Compare
Compares live ‘feels-like’ conditions across multiple locations at once, ranked, with asynchronous per-location loading.
BundleBuilder
Creates the structure of a court bundle — a case title page, a numbered index, and a printable divider for each exhibit, in the order you set. Print them and slot your documents behind each divider. Everything runs in the browser; built for people representing themselves in court.
