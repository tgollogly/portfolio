Test Bench

A small typed validation library — UK postcodes, sort codes, IBAN checksums, dates, money — together with the unit tests that prove it works. Both are TypeScript, and both run in this page. Edit the code, or press Introduce a bug and watch the suite go red and tell you exactly which case broke.

The types are stripped in the browser by Babel so the code can execute here. Full type-checking is a build step (tsc --noEmit) and is not what this page is showing — what it is showing is the test suite actually running.
